# copilot_remap
AHK script to remap tho Copilot key back to a RIGHT CTRL key  


How to use.
1. Install AutoHotKey which can be found at https://www.autohotkey.com/download/
2. Requires AHK V2
3. run copilot_remap.ahk by double clicking it
4. place in your startup folder if you want to make this start when booting up.

You can compile it using AhkToExe included in the ahk installation

I included a compiled version copilot_remap.exe
